# Mongo-Scraper
Mongo-Scraper is a dog news application that scrapes and displays articles from Dogtime for users to save to their favorites and add notes.
<br />

# Technology Used
* Express
* Express-Handlebars
* MongoDB
* Mongoose
* Axios
* Cheerio
